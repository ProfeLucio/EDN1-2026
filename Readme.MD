# 🚀 Estructuras de Datos en [Lenguaje, ej: Java / Python / C++]

¡Bienvenido! Este repositorio contiene implementaciones desde cero de las estructuras de datos más fundamentales y utilizadas en la computación. Ideal para repasar fundamentos, prepararse para entrevistas técnicas o como material de referencia.

---

## 📌 Contenido

El repositorio está organizado por tipo de estructura:

### 1. Estructuras Lineales
*   **Listas Enlazadas:** [Simple], [Doble], [Circular].
*   **Pilas (Stacks):** Implementación basada en arreglos y nodos.
*   **Colas (Queues):** Colas simples, circulares y de prioridad.

### 2. Estructuras No Lineales
*   **Árboles (Trees):** Árbol Binario de Búsqueda (BST), AVL, Heaps.
*   **Grafos:** Representación por matriz y lista de adyacencia (BFS/DFS).
*   **Tablas Hash:** Manejo de colisiones mediante encadenamiento y direccionamiento abierto.
